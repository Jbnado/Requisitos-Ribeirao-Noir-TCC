# Ribeirão Noir: O Roubo da Estátua de São Sebastião

Repositório contendo os requisitos funcionais e não funcionais do jogo "Ribeirão Noir: O Roubo da Estátua de São Sebastião" — um trabalho de TCC.

## Objetivo

Manter e documentar os requisitos do jogo, organizados para facilitar implementação, revisão e uso em entregas acadêmicas.

## Conteúdo

- `requisitos.md` — Documento principal com os Requisitos Funcionais (RF01..RF20) e Requisitos Não Funcionais (RNF01..RNF12).
- `citation.cff` — Metadados de citação do trabalho (autor, título, versão, data de liberação).

## Estrutura e formato

O arquivo `requisitos.md` está organizado em seções numeradas para grupos de requisitos (por exemplo, "Sistema de Rolagem de Dados") e cada requisito individual está como item com seu código em negrito (por exemplo, **RF01**) para facilitar referências.
